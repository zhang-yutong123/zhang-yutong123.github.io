# Zhang Yutong Personal Homepage

Static GitHub Pages website for Zhang Yutong.

## How to Use

1. Create a GitHub account.
2. Create a new public repository named:

```text
zhang-yutong123.github.io
```

3. Copy these files into that repository.
4. Replace all `zhang-yutong123`, `YOUR_EMAIL`, and `YOUR_LINKEDIN_SLUG` placeholders.
5. Commit and push.
6. Open:

```text
https://zhang-yutong123.github.io
```

This site uses plain HTML and CSS, so it does not need Node.js, npm, React, or any build step.
